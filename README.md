# Sudoku Online — Full Stack Web Application

Sudoku Online is a full-stack web application that provides an interactive browser-based Sudoku experience with secure authentication, game progress saving, leaderboard ranking, and real-time multiplayer support.

Built using **React, Node.js, Express, PostgreSQL, Prisma, Redis, and Socket.IO**, the platform allows users to play Sudoku smoothly across devices without installing any app.

## Features

* User Authentication using JWT
* Sudoku Puzzle Generation (Easy / Medium / Hard)
* Interactive 9x9 Sudoku Grid
* Notes Mode (Pencil Marks)
* Hint System and Mistake Counter
* Save and Resume Game Progress
* Global Leaderboard
* Real-Time Multiplayer Rooms
* Live Progress Tracking
* Room Chat System

## Tech Stack

### Frontend

* React

### Backend
